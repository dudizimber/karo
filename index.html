Karo (Kubernetes Alert Reaction Operator) Helm Repository

This is the official Helm repository for Karo.

Add Repository

helm repo add karo https://dudizimber.github.io/karo/
helm repo update

Install Chart

helm install my-karo karo/karo

Available Charts

  • index.yaml - Helm repository index
  • karo-v0.2.0.tgz

For more information, visit the project repository.
